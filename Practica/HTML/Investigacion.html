Invesigación de elementos basicos html

<-!DOCTYPE html>:

Define la versión de HTML que se está utilizando en el documento.

<-html>:

Es el elemento raíz de un documento HTML y engloba todo el contenido de la página.

<-head>:

Contiene información sobre el documento HTML, como el título de la página, enlaces a hojas de estilo, metadatos y scripts.

<-title>:

Define el título de la página web, que se muestra en la barra de título del navegador o en las pestañas del navegador.

<-body>:

Contiene el contenido principal de la página web, como texto, imágenes, videos, enlaces, formularios, etc.

<-header>:

Se utiliza para definir el encabezado de una sección o página web. Puede contener logotipos, menús de navegación, etc.

<-footer>:

Define el pie de página de una sección o página web. Suele contener información de contacto, enlaces adicionales, derechos de autor, etc.

<-nav>:

Se utiliza para definir una sección de navegación en la página web. Contiene enlaces a diferentes partes del sitio web.

<-section>:

Define una sección temática en una página web. Puede contener varios elementos como encabezados, párrafos, listas, etc.

<-article>:

Se utiliza para definir un contenido independiente y autónomo en una página web, como una publicación de blog o un artículo de noticias.

<-div>:

Es un contenedor genérico que se utiliza para agrupar elementos y aplicar estilos o scripts a un conjunto de elementos.
